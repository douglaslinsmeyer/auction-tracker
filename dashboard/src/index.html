Help & Documentation

About Auction Tracker

Auction Tracker is a comprehensive monitoring and bidding system for nellisauction.com. It consists of three integrated components working together to help you track and participate in auctions effectively.

🌐 Chrome Extension

Integrates directly with nellisauction.com to monitor auctions and place bids

🖥️ Dashboard

Central control panel to view and manage all monitored auctions

⚙️ Backend Server

Handles bidding logic, real-time updates, and data persistence

Getting Started

  1. 1

    Install the Chrome Extension

    Load the extension from the 'extension' folder in Chrome's developer mode

  2. 2

    Start the Backend Server

    Run 'docker-compose up' or 'npm run dev' in the backend directory

  3. 3

    Access the Dashboard

    Open http://localhost:3001 to view the dashboard

  4. 4

    Login to Nellis Auction

    Navigate to nellisauction.com and login with your account

  5. 5

    Start Monitoring

    Click the extension icon on any auction page to begin monitoring

How to Use

Monitoring Auctions

  • Navigate to any auction page on nellisauction.com
  • Click the Auction Tracker extension icon
  • Set your maximum bid amount
  • Choose a bidding strategy (Auto or Snipe)
  • Click "Start Monitoring" to begin tracking

Bidding Strategies

Auto (Incremental)
Automatically places bids whenever you're outbid, up to your maximum amount
Snipe
Waits until the final seconds to place a single bid, reducing bidding wars

Managing Auctions

  • View all monitored auctions in the dashboard
  • Edit bid amounts and strategies in real-time
  • Enable/disable auto-bidding per auction
  • Stop monitoring auctions you're no longer interested in
  • View auction history and bidding activity
